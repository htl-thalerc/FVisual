Clone Git Repo for Explorer:

Open Git Bash --> Navigate to your folder where you want to clone it --> write: 
git clone https://github.com/htl-thalerc/FVisual.git

Add files to Git Repo
move to FVisual with cd/cd ..

write: git add . //Adds all your changes
write: git commit -m "<Your commmit message>" //Saves local
write: git push //Pushes to master
